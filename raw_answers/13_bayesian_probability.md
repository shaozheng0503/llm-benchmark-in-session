**问题分析**

设事件：
- D = 真患病
- + = 检测阳性
- ¬D = 未患病

已知：
- 先验 P(D) = 0.01
- 真阳性率 P(+|D) = 0.99
- 真阴性率 P(−|¬D) = 0.99，故假阳性率 P(+|¬D) = 0.01

**应用贝叶斯公式**

P(D|+) = P(+|D) · P(D) / P(+)

其中 P(+) = P(+|D)·P(D) + P(+|¬D)·P(¬D)
            = 0.99 × 0.01 + 0.01 × 0.99
            = 0.0099 + 0.0099
            = 0.0198

P(D|+) = (0.99 × 0.01) / 0.0198 = 0.0099 / 0.0198 = **0.5**

**结论**：检测呈阳性的人，真正患病的概率约为 **50%**，远低于直觉上的 99%。

这是经典的**基率忽略（base rate neglect）**陷阱：因为疾病本身极罕见（1%），即使检测准确率高达 99%，假阳性在阳性结果中仍占一半。
